# PWA Icons

This directory should contain the following icon files for the PWA:

- `Icon-192.png` - 192x192px icon
- `Icon-512.png` - 512x512px icon
- `Icon-maskable-192.png` - 192x192px maskable icon
- `Icon-maskable-512.png` - 512x512px maskable icon

## Generating Icons

You can generate these icons from your app logo using tools like:
- https://www.pwabuilder.com/imageGenerator
- https://maskable.app/editor

## Icon Requirements

- **Standard icons**: Should have transparent background
- **Maskable icons**: Should have safe zone padding (20% on all sides)
- **Format**: PNG
- **Sizes**: 192x192 and 512x512 pixels

For now, placeholder icons are used. Replace them with actual branded icons before production deployment.
